# Portfolio-analysis---Diversification--Risk-analysis--performance-of-Investment-with-the-Excel 

Portfolio Analysis & Optimization with the help of Excel

 Project Overview
 
This project provides a comprehensive framework for analyzing investment portfolios. Using historical market data, it calculates key performance metrics, visualizes risk-return profiles, and performs Mean-Variance Optimization to identify the Efficient Frontier.

The goal is to help investors make data-driven decisions by quantifying the trade-offs between expected return and volatility.

#Key Features

Data Acquisition: Automated fetching of historical stock data via APIs (e.g., Yahoo Finance/yfinance).

Performance Metrics: Calculation of Annualized Returns, Volatility, and the Sharpe Ratio.

Risk Assessment: Analysis of Maximum Drawdown and Value at Risk (VaR).

Portfolio Optimization: * Monte Carlo Simulation: Generating thousands of random portfolios to find the optimal mix.

Efficient Frontier: Visualizing the set of optimal portfolios that offer the highest expected return for a defined level of risk.

Correlation Heatmaps: Identifying how different assets move in relation to one another to ensure true diversification.
